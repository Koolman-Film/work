About this mockup

S-N14 = drill-down per-employee payroll slip for Admin. Used for detailed review, manual override of fields, unlock + revision (after publish), and Owner read-only viewing.

Key UI elements

  • Employee card: avatar + meta + key info grid (base salary, dept, branch, etc.)
  • NetPay hero: gradient blue card with big tabular number + actions (Download PDF, Unlock revision)
  • Income breakdown card: all income sources with edit ✏ icons + override markers
  • Deduction breakdown card: all deductions with red minus + linked source detail
  • Audit timeline: chronological history including override events with before/after change pill

States

  • Published view: 🔒 locked · ✏ icons hidden · "Unlock for revision" button shown
  • Draft view: editable · ✏ icons visible · "Save & approve" button
