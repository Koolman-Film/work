วิธีลองใช้

  • คลิก state pill เพื่อสลับ state · หรือ พิมพ์ password จริง ใน input เพื่อดู strength bar เปลี่ยน live
  • ลองพิมพ์ 123 · password · Pass1234 · P@ssw0rd!2026 เพื่อดู 4 ระดับ
  • พิมพ์ confirm ไม่ตรง → match status เป็นแดง
  • เมื่อ password valid + match → submit enable

States covered

  • Default — token valid, form empty, submit disabled
  • Typing weak — strength bar เริ่มแดง · checks ยังไม่ครบ
  • Typing strong — strength bar เขียวเต็ม · checks ทุกข้อ
  • Confirm mismatch — match status "ไม่ตรงกัน" สีแดง
  • Valid — ทุกอย่าง pass · submit enabled
  • Loading — submitting · spinner
  • Success — toast + UX redirect ไป dashboard
  • Invalid token (X-A3) — full-page error state แทน form · "ลิงก์เชิญหมดอายุ — ติดต่อแอดมิน"
💡 Implementation: Strength calc แบบง่ายๆ — score ตาม length (≥8) + has digit + has uppercase + has special char. ไม่ต้องใช้ zxcvbn library V1 (overkill สำหรับ HR app).
