Form pattern showcase

This mockup demonstrates the standard form pattern for employee app:

  • Topbar: back arrow + form title + spacer (no avatar/bell to focus on form)
  • Field stack: single column, generous spacing (Soft Modern padding)
  • Required marker: red * after label
  • Live counter: char count under textarea
  • Smart hint: "รวม X วัน (ไม่รวมวันหยุด)" auto-calculates from dates
  • File upload: drag-drop area + file preview when uploaded
  • Action row: primary submit + secondary cancel, fixed at bottom of form

States covered

  • Default — empty form, submit disabled until valid
  • Filled valid — all fields filled correctly, submit enabled
  • With attachment — file uploaded, preview shown
  • Error invalid dates — endDate before startDate
  • Error file too large — > 5MB
  • Loading — submitting · spinner
  • Success — toast T-E1 + redirect
  • Discard modal — confirms before exit if dirty
