วิธีใช้

1. คลิก option ด้านบน → เปิดใน tab ใหม่
2. ดู mockup จริง (login, dashboard, table, components) — ทั้ง light + dark mode
3. เปรียบเทียบ tabs side-by-side
4. เลือก direction → tell me ว่าจะใช้ option ไหน → ผมเขียน design-system.md ตามที่เลือก

เลือกได้ไหม?

✅ เลือก 1 option ตรงๆ — ผมเอาไป build full design system
✅ Mix-match — เช่น "color จาก option 1 + font จาก option 4" — บอกได้
✅ Tweak — เช่น "option 1 แต่ accent เปลี่ยนจาก amber → red" — บอก hex code ได้
✅ None of these — บอก reference (เว็บที่ชอบ) → ผมหาทางเลือกใหม่
