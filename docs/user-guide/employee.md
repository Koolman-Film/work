# คู่มือพนักงาน — Koolman Work

ระบบ HR สำหรับพนักงาน Koolman ใช้งานผ่านแอป **LINE** ทั้งหมด ไม่ต้องดาวน์โหลดแอปใหม่

---

## 🎬 ครั้งแรก: เชื่อมบัญชี LINE กับระบบ (ทำครั้งเดียว)

1. **รับลิงก์จากแอดมิน** — แอดมินจะส่งลิงก์ให้คุณทาง LINE / SMS / อีเมล หรือพิมพ์ QR Code ให้สแกน
2. **กดลิงก์** หรือสแกน QR ด้วยกล้องโทรศัพท์
3. หน้าเว็บจะเด้งขึ้นมา → กดปุ่ม **"เปิดในแอป LINE →"**
4. แอป LINE จะเปิด → กดยอมรับ **"เพิ่ม Koolman Work เป็นเพื่อน"**
5. รอ ~3 วินาที → ระบบจะเชื่อมบัญชีให้อัตโนมัติ → เข้าหน้าเช็คอิน

> 💡 ลิงก์จับคู่มีอายุ **24 ชั่วโมง** หากหมดอายุ ติดต่อแอดมินเพื่อขอลิงก์ใหม่

---

## 📍 เช็คอิน-เช็คเอาท์ประจำวัน

### เปิด LIFF
1. เปิดแอป LINE → เข้าแชท **Koolman Work** (ปักหมุดไว้ที่ด้านบนของรายการแชท)
2. กดปุ่ม **"📍 เช็คอิน"** ที่เมนูล่าง (Rich Menu)

### เช็คอินเข้างาน
1. กดปุ่มสีน้ำเงิน **"เช็คอินเข้างาน"**
2. แอปจะขออนุญาตเข้าถึงตำแหน่งที่ตั้ง → กด **อนุญาต**
3. หากสาขาของคุณต้องการเซลฟี่ → กล้องจะเปิด → ถ่ายรูปตัวเอง → กด **ใช้รูปนี้**
4. ระบบจะตรวจสอบ → ขึ้นข้อความ **"เช็คอินสำเร็จ"** หากอยู่ในรัศมีสาขา

### เช็คเอาท์ออกงาน
1. กลับมาที่หน้าเช็คอิน
2. กดปุ่ม **"เช็คเอาท์"**

> ⚠️ ถ้าระบบบอก **"เช็คอินบันทึกแล้ว แต่ต้องตรวจสอบ"** หมายความว่าตำแหน่งของคุณอยู่นอกรัศมีสาขา — แอดมินจะตรวจสอบให้ภายหลัง คุณยังเช็คอินไปทำงานได้ปกติ

---

## 📅 ส่งคำขอลา

1. ที่หน้าเช็คอิน → กดปุ่ม **"📅 คำขอลา"**
2. กดปุ่ม **"+ ส่งคำขอ"**
3. กรอกข้อมูล:
   - **ประเภทการลา:** ลาป่วย / ลากิจ / ลาพักร้อน
   - **วันเริ่มต้น + วันสิ้นสุด:** เลือกจากปฏิทิน
   - **เหตุผล:** อย่างน้อย 4 ตัวอักษร
   - **แนบไฟล์** (ไม่บังคับ): เช่น ใบรับรองแพทย์
4. กด **"ส่งคำขอ"**
5. แอดมินจะได้รับการแจ้งเตือนและตรวจสอบให้

### ดูสถานะ + ยกเลิก
- หน้า **"คำขอลาของฉัน"** จะแสดงคำขอทั้งหมด
- หากยังไม่ได้รับการอนุมัติ คุณสามารถกด **"ยกเลิก"** ได้
- เมื่อแอดมินอนุมัติ/ปฏิเสธ คุณจะได้รับการแจ้งเตือนทาง LINE

---

## 💰 ขอเบิกเงินล่วงหน้า

1. ที่หน้าเช็คอิน → กดปุ่ม **"💰 ขอเบิก"**
2. กดปุ่ม **"+ ส่งคำขอ"**
3. กรอกจำนวนเงิน (สูงสุด ฿100,000 ต่อครั้ง)
4. กด **"ส่งคำขอ"**
5. แอดมินจะอนุมัติ + แนบใบเสร็จเมื่อจ่ายเงิน

> 💡 ระบบอนุญาตให้มีคำขอเบิกที่ **รออนุมัติได้ครั้งเดียว** — หากต้องการเปลี่ยนจำนวน ให้ยกเลิกคำขอเดิมก่อน

---

## 🗓️ ดูปฏิทินทีม

1. ที่หน้าเช็คอิน → กดปุ่ม **"🗓️ ปฏิทินทีม"**
2. ดูว่าใครในสาขาเดียวกับคุณลาในวันไหน
3. **สีฟ้า** = อนุมัติแล้ว, **สีเหลือง** = รออนุมัติ
4. กดวันเพื่อดูรายละเอียดของวันนั้น

---

## 👤 แก้ไขโปรไฟล์

1. ที่หน้าเช็คอิน → กดปุ่ม **"👤 โปรไฟล์"**
2. แก้ไขข้อมูลส่วนตัว: เบอร์โทร, อีเมล, ที่อยู่, ผู้ติดต่อฉุกเฉิน, ชื่อเล่น
3. กด **"บันทึก"**

> ⚠️ ข้อมูลงาน (สาขา, แผนก, เงินเดือน, วันเริ่มงาน) แก้ได้เฉพาะแอดมิน — ติดต่อแอดมินหากต้องเปลี่ยน

---

## ❓ ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|---|---|
| GPS ไม่ทำงาน / ระบุตำแหน่งไม่ได้ | เปิด Location ในการตั้งค่าโทรศัพท์ + อนุญาตให้ LINE ใช้ตำแหน่ง |
| เช็คอินถูกตั้งว่า "ต้องตรวจสอบ" | คุณอยู่นอกรัศมีสาขา (150 ม.) — รอแอดมินตรวจสอบ |
| "ขาดลิงก์การจับคู่" หลังกดลิงก์ | ลิงก์อาจหมดอายุ → ขอลิงก์ใหม่จากแอดมิน |
| ไม่ได้รับการแจ้งเตือน LINE | ตรวจสอบว่าได้เพิ่ม Koolman Work เป็นเพื่อนแล้ว |
| เปิดลิงก์แล้วเด้งกลับหน้า Login | เปิดลิงก์ผ่านแอป LINE เท่านั้น ไม่ใช่ Chrome / Safari |
