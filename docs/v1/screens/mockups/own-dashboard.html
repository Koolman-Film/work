About this mockup

S-O1 = Owner's daily overview screen. Information-rich, no actions (read-only). Owner sees salary data (q12 confirmed by customer).

Layout

  • Lighter sidebar — only 4 items + ⭐ Owner role badge (orange tint)
  • Owner avatar — accent orange (vs Admin's blue) — visual role distinction
  • 5 KPI cards — hero card spans 2 cols (NetPay total) + 4 smaller cards
  • Charts row — 6-month trend (left, 2/3 width) + deduction donut breakdown (right, 1/3)
  • 2-col section — Top 10 leave/late + Override alerts (with EmpID + before/after values)
  • Today on leave — full-width section listing who's currently on leave/late

Key features

  • Hero KPI — gradient blue with NetPay total (most relevant for Owner)
  • SVG charts — inline SVG (real impl uses recharts) — line + area + donut shown
  • Override alerts — auto-filtered audit (override-type only) for Owner attention
  • Top 10 ranking — gold highlight on #1 (visual hierarchy)
