วิธีลองใช้

  • คลิก state pill ด้านบนเพื่อสลับ state ของ login form
  • Hover/focus input เพื่อดู soft border + brand glow ring
  • Toggle dark mode ที่ topbar เพื่อดู dark variants
  • Resize หน้าต่างต่ำกว่า 1100px เพื่อดู mobile-only layout

States covered

  • Default — empty form, submit ปกติ
  • Filled — user typed values, ปุ่มยังเปิด
  • Focused — input มี ring แบบ shadow-glow
  • Caps Lock detected — hint chip ใต้ password field
  • Loading — ปุ่มมี spinner + "กำลังเข้าสู่ระบบ..." + inputs disabled
  • Error toast — bottom-right · "เบอร์โทรหรือรหัสผ่านไม่ถูกต้อง" (T-A2)
  • Success → redirect — toast info "ส่ง OTP ไปยัง..." (T-A1) — UX จะ redirect ไป S-A2
💡 Pro tip: ใน real app — ปุ่มใช้ <Button variant="primary" size="md" full-width> จาก shadcn customized ตาม design-system. Form ใช้ React Hook Form + Zod resolver.
