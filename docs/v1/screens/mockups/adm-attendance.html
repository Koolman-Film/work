About this mockup

Attendance records table = primary data view for Admin. Key V1 differentiator from competitors: manual override of DeductionAmount with audit trail.

Key UI elements

  • Summary cards: 5 colored cards showing month aggregates (work/leave/absent/late/deduct)
  • Source tags: column shows where record came from — SCANNER (excel), MANUAL (admin keyed), AUTO (from leave approval)
  • Type badges: color-coded (green work, blue sick, purple personal, orange late, red absent, gray noscan)
  • Override indicator: ✏ icon + orange ring on amount cell when manual override
  • Override modal (M-N3): shows current auto-calc → editable amount + required reason · audit log generated on save
  • Filter chip: "⚠ Override only" quick filter for Owner review

States covered

  • List view — table with mixed sources (auto/scanner/manual)
  • Override modal — current amount displayed, editable, required reason
  • Saved toast — confirms override + audit logged
