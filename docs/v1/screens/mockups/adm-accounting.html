About this mockup

S-N15 = monthly accounting export. Generates CSV format compatible with PEAK Account (Thai accounting software customer uses).

Layout

  • 2-column layout: filter card (left) + preview/actions (right)
  • Recent exports list — full width below for re-download

Key UI elements

  • PEAK badge in subtitle — visual indicator of target system
  • Filter card: month selector with Published indicator (✓ ready), group selector (ทุกกลุ่ม / specific), branch, format
  • Preview stats: 3 KPI cards (employee count, total income, total NetPay)
  • By-Group breakdown — visual rows showing AccountingGroup totals
  • Export buttons: 2 large buttons side-by-side — primary (PEAK CSV) + secondary (Excel summary) — both show filename below for clarity
  • Recent exports: last 4 with re-download action · monospace filename for tech feel

States

  • Default: filters set, preview + buttons ready
  • Exporting: button disabled with spinner
  • Downloaded: success toast (T-N8) with filename
