# คู่มือเจ้าของ — Koolman Work

สำหรับผู้บริหาร / เจ้าของกิจการ ที่ต้องการดูภาพรวมโดยไม่ต้องจัดการรายละเอียดเอง

> 💡 บัญชี **Owner** เป็นบัญชี **อ่านอย่างเดียว** (read-only) — ดูข้อมูลได้ทั้งหมด แต่ไม่สามารถแก้ไข อนุมัติ หรือลบ ได้ ทุกการเปลี่ยนแปลงต้องผ่าน Admin

---

## 🔐 เข้าสู่ระบบ

1. เปิดเว็บ: **https://work.kool-man.com**
2. กรอกอีเมล (ที่ลงทะเบียนเป็น Owner) + รหัสผ่าน
3. ระบบจะนำคุณไปหน้า **`/owner`** อัตโนมัติ

---

## 📊 หน้าหลัก Owner

### KPI สรุปประจำวัน
- **คำขอลา รออนุมัติ** — มีกี่คำขอที่ Admin ยังไม่ได้ตัดสินใจ
- **คำขอเบิก รออนุมัติ** — เหมือนกัน ฝั่งเงินสด
- **เช็คอินวันนี้** — กี่คนมาทำงาน
- **ลาวันนี้** — กี่คนลา

### กิจกรรมล่าสุด
รายการเหตุการณ์สำคัญที่เกิดขึ้นล่าสุด — เช่น พนักงานคนใหม่ส่งคำขอ, แอดมินอนุมัติคำขอ ฯลฯ

---

## 🔔 การแจ้งเตือน

ระฆัง 🔔 ด้านบนขวาทำงานเหมือนกับ Admin:
- รับการแจ้งเตือนเมื่อพนักงานส่งคำขอใหม่
- รับการแจ้งเตือนเมื่อมีเช็คอินผิดปกติ
- รับการแจ้งเตือนเมื่อพนักงานทดลองงานใกล้ครบ 4 เดือน

แม้คุณจะไม่ได้เป็นคนตัดสินใจ แต่การรับการแจ้งเตือนช่วยให้รู้ว่ามีอะไรเกิดขึ้นในธุรกิจ

---

## 🤝 ทำงานร่วมกับ Admin

หากคุณต้องการให้แก้ไขข้อมูล / อนุมัติคำขอ / เปลี่ยนการตั้งค่า:
1. ติดต่อ Admin โดยตรง (โทร / LINE / เดินไปบอก)
2. Admin จะดำเนินการให้
3. คุณดูผลลัพธ์ในหน้า Owner ได้

> 💡 หากคุณต้องทำหน้าที่ทั้ง Owner + Admin คนเดียว — ขอบัญชี Admin แยกอีกใบจาก Admin ระบบ (Phase-2 อาจรวมเป็นบัญชีเดียวที่มีทั้ง 2 สิทธิ์)

---

## 📅 หน้าอื่น ๆ (กำลังพัฒนา)

หน้าเหล่านี้จะมีให้ใช้งานใน **Phase 3** (ราวเดือนกรกฎาคม):
- **/owner/calendar** — ปฏิทินรวมของทั้งบริษัท (กรองตามสาขา/แผนก)
- **/owner/payroll/[month]** — ดูเงินเดือนรายเดือน (อ่านอย่างเดียว)
- **/owner/audit** — log การเปลี่ยนแปลงข้อมูลทั้งหมด
