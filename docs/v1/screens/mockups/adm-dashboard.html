About this mockup

Admin Dashboard = daily home for Admin role. Shows KPIs, pending items, alerts, trends. Desktop-first layout with sidebar nav.

Layout

  • Browser frame: mock เพื่อแสดงว่าเป็น web app (URL bar = app.koolman-hr.com/admin/dashboard)
  • Sidebar (220px): 9 main items + bottom profile/sign-out · active state มี left-border + bg highlight
  • Sidebar badges: red count for pending items (8 leave / 3 advance) · warning ⚠ for unpublished payroll
  • Main content: alert → KPI row (4 cards) → 2-column section → trend chart
  • Topbar in main: breadcrumb + 🔔 + avatar

Key UI elements

  • Alert banner: warning style with brand orange accent, action button right-aligned
  • KPI cards: label · big tabular number · meta line (with trend arrow + warn color)
  • Pending list: avatar + name + detail + time-ago, hover highlight
  • Trend chart: SVG inline (real impl uses recharts)

Mobile responsive

Below 768px: sidebar collapses to drawer (hamburger trigger), KPI grid becomes 2-col instead of 4-col.
