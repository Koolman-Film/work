วิธีลองใช้

  • คลิก state pill เพื่อสลับ state ต่างๆ
  • ลอง พิมพ์ตัวเลข ใน cell — auto-advance ไป cell ถัดไป
  • ลอง backspace — กลับ cell ก่อนหน้า
  • ลอง paste 6 digits ใน cell แรก — auto-fills ทุก cell
  • เมื่อพิมพ์ครบ 6 → submit button enable

States covered

  • Empty — เพิ่งเข้ามา · auto-focus cell แรก · countdown 1:00 เริ่มเดิน
  • Typing (3/6) — กำลังพิมพ์ · cells filled มีพื้นหลัง primary-50
  • Filled — ครบ 6 หลัก · submit button enabled
  • Loading — submitting · cells disabled · spinner ใน button
  • Wrong code (X-A2) — toast error + cells flash red shake animation + clear · focus cell แรก
  • Resend ready — countdown หมด · "ส่งรหัสอีกครั้ง" button แทนที่ countdown
  • Success — toast success "ยืนยันสำเร็จ" + UX redirect ไป dashboard ตาม role
💡 Implementation: ใช้ shadcn <InputOTP> component หรือ custom 6-cell. Auto-focus next cell ด้วย onChange handler. Paste support ผ่าน onPaste.
