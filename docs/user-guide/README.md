# คู่มือผู้ใช้ Koolman Work

คู่มือการใช้งานระบบ HR ภายในของ Koolman แบ่งตามบทบาทผู้ใช้

| บทบาท | คู่มือ | สำหรับใคร |
|---|---|---|
| 👤 **พนักงาน** (Employee) | [employee.md](./employee.md) | ทุกคนที่ทำงานใน Koolman ใช้ผ่านแอป LINE |
| 🛠️ **แอดมิน** (Admin) | [admin.md](./admin.md) | HR, บัญชี, ผู้จัดการสาขา ใช้ผ่านเว็บ |
| 👁️ **เจ้าของ** (Owner) | [owner.md](./owner.md) | ผู้บริหาร — ดูภาพรวมอย่างเดียว ไม่แก้ไขข้อมูล |

## สิ่งที่ต้องเตรียม

- พนักงาน: โทรศัพท์ที่ติดตั้งแอป LINE (iOS / Android)
- แอดมิน / เจ้าของ: คอมพิวเตอร์หรือมือถือพร้อมเว็บเบราว์เซอร์

## URL หลัก

- **เว็บแอดมิน:** https://work.kool-man.com
- **LIFF พนักงาน:** เปิดผ่านแอป LINE (จากเมนูล่างของ Koolman Work หรือลิงก์ที่แอดมินส่ง)

## ช่วยเหลือ + ติดต่อ

หากพบปัญหาในการใช้งาน:
1. ลองทำตามคู่มือของบทบาทตัวเอง
2. ติดต่อแอดมินที่ดูแลระบบ
3. หากเป็นปัญหาเทคนิคที่แอดมินแก้ไม่ได้ ติดต่อทีมพัฒนา
