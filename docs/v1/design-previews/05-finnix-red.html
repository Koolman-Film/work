Brand red as accent only — onyx + white dominate, red highlights key actions

Combo: onyx dominant + red-600 accent · Anuphan · premium automotive (restrained)

📐 Red usage rules (constrained)

  • Logo & brand mark Logo square + thin top stripe — signature ที่จดจำได้
  • Brand-action buttons เท่านั้น เช่น "ส่งคำขอลา" / "ขอเบิกเงิน" — main CTA สำคัญต่อ business — ไม่ใช่ทุกปุ่ม
  • Status: Rejected / Absent หมายถึง "ปัญหา" — สื่อ semantic อยู่แล้ว
  • Hero accent Subtle border-left + radial glow — ไม่ใช่ background
  • NOT used หัวเรื่อง, link ปกติ, pending status, hover state, table rows

⭐ ทำไมเลือก Theme นี้ (refined)

  • Brand-aligned แต่ไม่ overwhelming — Finnix red ปรากฏแค่ที่จำเป็น
  • Onyx + white = 90% ของ screen → eye comfort ใช้นานๆ ได้
  • Red ใช้ semantic — ปุ่มแดง = "ส่ง" / status แดง = "ปัญหา" — meaning ชัดเจน
  • High contrast — ตัวเลข/ตารางอ่านง่าย (black on white)
  • Premium feel — minimalist + restraint = high-end automotive (think Apple Pro)
  • Marketing alignment — ยังคง Finnix DNA โดยไม่ตะโกน
  • No red fatigue — แก้ปัญหาของ version แรก

Rule of thumb: red ≤ 10% ของ pixel ใน screen ใดๆ — ที่เหลือ onyx + white
