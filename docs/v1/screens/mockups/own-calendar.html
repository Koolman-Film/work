About this mockup

S-O2 = Owner's most-used screen. Full company-wide calendar — see who's on leave/late/absent at a glance. Click any day → drawer with names + drill-down links.

Layout

  • Lighter sidebar — only 4 items (vs Admin's 9) · "⭐ Owner" badge
  • Toolbar: month nav + filter chips (department, branch, type) + view toggle (Day/Week/Month)
  • Month grid: 7-col, each cell shows day number + event chips + summary
  • Today highlighted — primary-50 bg + bold day number
  • Event chips — color-coded by type, max 3 visible per cell, "+N more" if overflow
  • Day drawer (D-O1) — opens on cell click · grouped by event type · drill-down "ดูรายละเอียด" links

Color legend

  • 🟥 ขาด — red bg/text
  • 🟦 ลาป่วย — blue
  • 🟪 ลากิจ — purple
  • 🟩 พักร้อน — green
  • 🟧 สาย — orange
  • ⬜ ไม่สแกน — gray

Performance considerations

  • Server pre-aggregates daily counts per cell — table-scan via DB index, not row-by-row JS calc
  • Initial query: 30 days × department/branch filter
  • Day drawer fetched on click (not pre-loaded all days)
  • Realtime subscription: not needed for V1 (Owner refreshes manually)
