# คู่มือแอดมิน — Koolman Work

สำหรับ HR, บัญชี, ผู้จัดการสาขา ที่ดูแลระบบประจำวัน เข้าใช้งานผ่านเว็บ

---

## 🔐 เข้าสู่ระบบ

1. เปิดเว็บ: **https://work.kool-man.com**
2. กรอกอีเมลและรหัสผ่านที่ได้รับจากผู้ดูแลระบบ
3. ระบบจะนำคุณไปหน้า **"ภาพรวม"** (Dashboard)

> 💡 ลืมรหัสผ่าน? กด **"ลืมรหัสผ่าน?"** → กรอกอีเมล → รับลิงก์รีเซ็ตทางอีเมล

---

## 📊 หน้าหลัก (Dashboard)

เปิด `/admin` หรือกด **"หน้าหลัก"** ในเมนูซ้าย

**4 การ์ด KPI ด้านบน:**
- **คำขอลา รออนุมัติ** — กดเพื่อไปหน้าตรวจคำขอลา
- **คำขอเบิก รออนุมัติ** — กดเพื่อไปหน้าตรวจคำขอเบิก
- **เช็คอินวันนี้** — จำนวนคนที่เช็คอินแล้ว
- **ยังไม่เช็คอินวันนี้** — จำนวนคนที่ควรมาทำงานแต่ยังไม่เช็คอิน (สีแดง = ต้องติดตาม)

**สองช่องด้านล่าง:**
- **คำขอที่รอดำเนินการ** — รายการ 5 อันล่าสุดของคำขอลา + คำขอเบิก
- **ลาวันนี้** — รายชื่อพนักงานที่ลาวันนี้

---

## 👥 จัดการพนักงาน

**เมนูซ้าย → "พนักงาน"**

### เพิ่มพนักงานใหม่
1. กดปุ่ม **"+ เพิ่มพนักงาน"** มุมขวาบน
2. กรอกข้อมูล: ชื่อ-นามสกุล, ชื่อเล่น, สาขา, แผนก, เงินเดือน, วันเริ่มงาน ฯลฯ
3. กด **"บันทึก"**

### สร้างลิงก์เชื่อม LINE
1. เข้าหน้าแก้ไขพนักงาน
2. ในการ์ด **"การเชื่อม LINE"** → กดปุ่ม **"📩 สร้างลิงก์ LINE"**
3. ระบบจะแสดงลิงก์ + QR Code (อายุ 24 ชม.)
4. ส่งลิงก์ให้พนักงาน หรือพิมพ์ QR Code ให้สแกน
5. เมื่อพนักงานเชื่อมเสร็จ การ์ดจะเปลี่ยนเป็นสีเขียว ✅

> ⚠️ ลิงก์เก่าจะใช้ไม่ได้เมื่อกด **"🔄 สร้างลิงก์ใหม่"**

---

## 📅 ตรวจสอบคำขอลา

**เมนูซ้าย → "คำขอลา"**

- หน้านี้จะแสดงคำขอที่ **รออนุมัติ** เป็นค่าเริ่มต้น
- กรองตามสถานะด้วย Chips ด้านบน (ทั้งหมด / อนุมัติแล้ว / ไม่อนุมัติ)
- คลิกที่แต่ละคำขอเพื่อดูรายละเอียด:
   - ช่วงวันที่ + จำนวนวันทำงาน (ระบบคำนวณ ตัดวันอาทิตย์ + วันหยุดให้)
   - ไฟล์แนบ (เช่น ใบรับรองแพทย์)
   - ปุ่ม **อนุมัติ** / **ไม่อนุมัติ** พร้อมช่องกรอกหมายเหตุ
- เมื่ออนุมัติ ระบบจะสร้างรายการ Attendance (`OnLeave`) สำหรับแต่ละวันทำงานในช่วงให้อัตโนมัติ และส่ง LINE แจ้งพนักงาน

---

## 💰 ตรวจสอบคำขอเบิก

**เมนูซ้าย → "คำขอเบิก"**

1. คลิกคำขอที่รออนุมัติ
2. **อัปโหลดไฟล์ใบเสร็จ** (บังคับก่อนกดอนุมัติ)
3. กด **"อนุมัติ"** → ระบบส่ง LINE แจ้งพนักงาน
4. หรือกด **"ไม่อนุมัติ"** + กรอกเหตุผล

---

## ⏰ ดูการลงเวลา

**เมนูซ้าย → "ลงเวลา"**

### หน้าหลัก: ประวัติการลงเวลา (`/admin/attendance`)
- ดูรายการเช็คอิน / ลา / ขาด / สาย ของเดือนที่เลือก
- กรองตาม **เดือน + พนักงาน + ประเภท**
- รองรับ 200 รายการต่อหน้า — ใช้ตัวกรองเพื่อแคบลงหากเกิน

### ปุ่มลัดด้านบน
- **+ คีย์มือ** — บันทึก ขาด/สาย/ออกก่อน ด้วยตนเอง
- **🔴 วันนี้ (live)** — ดูเช็คอินสด ๆ ของวันนี้
- **⚠️ ต้องตรวจสอบ** — เช็คอินที่อยู่นอกรัศมีสาขา

### บันทึกขาด/สาย/ออกก่อนด้วยตนเอง (`/admin/attendance/manual`)
ใช้เมื่อพนักงานไม่สามารถเช็คอินด้วย LINE ได้:
1. เลือกพนักงาน + วันที่ + ประเภท (ขาดงาน / มาสาย / ออกก่อน)
2. หากเป็น "มาสาย" หรือ "ออกก่อน" — กรอกจำนวนนาที
3. กรอกหมายเหตุ (ไม่บังคับ)
4. กด **"บันทึก"**

### ตรวจสอบ Disputed (`/admin/attendance/disputed`)
- เช็คอินที่ระบบตั้งว่า **"ต้องตรวจสอบ"** เพราะอยู่นอกรัศมีสาขา
- ดูรูปเซลฟี่ + ตำแหน่งบนแผนที่
- กด **อนุมัติ** (ยอมรับเป็น Confirmed) หรือ **ไม่อนุมัติ** (ปฏิเสธ)

## ⚙️ ตั้งค่าระบบ

**เมนูซ้าย → "ตั้งค่า"**

| เมนู | สำหรับอะไร |
|---|---|
| **สาขา** | เพิ่ม/แก้ไขสาขา + ตำแหน่งบนแผนที่ (Geofence) + รัศมี |
| **แผนก** | เพิ่ม/แก้ไขแผนก เช่น ติดฟิล์ม, บัญชี, บริหาร |
| **กลุ่มบัญชี** | สำหรับการแยกค่าใช้จ่ายตามกลุ่ม (เชื่อมกับ PEAK ในอนาคต) |
| **ประเภทการลา** | เช่น ลาป่วย, ลากิจ, ลาพักร้อน + จำนวนวันต่อปี |
| **วันหยุด** | วันหยุดบริษัทประจำปี — ระบบจะตัดออกจากการคำนวณวันทำงานให้อัตโนมัติ |

> 💡 หากวันหยุดตรงกับวันอาทิตย์ (วันหยุดประจำสัปดาห์) ระบบจะ **เลื่อนวันหยุดชดเชยเป็นวันจันทร์** ให้อัตโนมัติตามกฎหมายแรงงาน

## 🔔 การแจ้งเตือน

ระฆัง 🔔 ด้านบนขวาจะแสดงการแจ้งเตือนใหม่:
- พนักงานส่งคำขอลา
- พนักงานส่งคำขอเบิก
- เช็คอินที่ต้องตรวจสอบ
- มีพนักงานยังไม่เช็คอิน (ทุกวัน 10:00 น.)
- พนักงานทดลองงานใกล้ครบ 4 เดือน (แจ้ง 7 วันล่วงหน้า)

กดที่การแจ้งเตือนเพื่อเปิดหน้าที่เกี่ยวข้อง หรือกด **"อ่านทั้งหมด"** เพื่อลบสีแดง
