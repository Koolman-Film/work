About this mockup

Pay slip detail = critical UX สำหรับ employee. คนใช้ดูเงินเดือน — accuracy + readability สำคัญที่สุด.

Key UI elements

  • Card hierarchy: slip body card (default bg) + NetPay separate emphasized card (gradient blue)
  • Tabular nums: all monetary values use IBM Plex Mono — แนวตั้งตรง
  • Section labels: uppercase tiny labels for "รายรับ" / "รายหัก" — visual chunks
  • Subtotal rows: bold + top border separator
  • Deduction styling: red color + auto "−" prefix on amounts
  • NetPay hero: blue gradient card, big tabular number, transfer info below
  • PDF action: top-right primary button — opens action sheet on mobile

States covered

  • Default view — full slip rendered
  • Mobile PDF action sheet (M-E3) — bottom sheet with download/share/cancel
  • After download — toast confirmation T-E5

Implementation notes

  • Money formatting: new Intl.NumberFormat('th-TH', { style: 'currency', currency: 'THB' })
  • PDF rendering: @react-pdf/renderer — same Thai font (IBM Plex Sans Thai bundle)
  • Storage: PDF cached in Supabase Storage, presigned URL TTL 7 days
  • Print styles: @media print hides topbar/nav, prints A4 portrait
